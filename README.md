# g3u8d
一个golang 写的简单的m3u8下载器

## 使用方法
```shell
   -r value, --retry value    设置重试次数 (default: 5)
   -t value, --timeout value  设置超时时间 (default: 15)
   -u value                   设置要下载的m3u8的url
   -p value                   设置要下载的m3u8的文件
   -f value                   设置保存的文件名称 (default: "file.mp4")
   -d value, --dir value      设置保存的目录
   -b value, --base value     设置base url
   --proxy value              设置使用的代理，格式如：http://localhost:3000
   --thread value             设置最大并发数 (default: 64)
   --help, -h                 show help (default: false)

```
